Projekte

Ich habe einige, kleinere OpenSource Projekte:

Chameleon

Python JavaScript

Chameleon ist ein Python-Modul zur Farb-Manipulation und Umwandlung diverser Farbmodelle. Derzeit ist es ein 1:1 Port der TinyColor JavaScript-Bibliothek und verfügt über den gleichen Funktionsumfang.

GitLab Repository

Slide Menu

CSS JavaScript

Slide Menu ist ein kleines jQuery Plugin das ein Standard CSS-Menü in eine multihierarchische Sidebar mit einem Gleit-Effekt verpackt. Die Animationen basieren auf CSS3 Transitions.

Das Menü hat viele Optionen und legt Methoden offen, um es programmatisch zu steuern.

GitHub Repository
